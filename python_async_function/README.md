# Python - Async

This project covers asynchronous programming in Python 3 using `asyncio`, `async`/`await` syntax, and the `random` module.

## Tasks

| File | Description |
|------|-------------|
| `0-basic_async_syntax.py` | Coroutine that waits a random delay and returns it |
| `1-concurrent_coroutines.py` | Runs multiple coroutines concurrently and returns sorted delays |
| `2-measure_runtime.py` | Measures average execution time of concurrent coroutines |
| `3-tasks.py` | Creates and returns an `asyncio.Task` |
| `4-tasks.py` | Runs multiple Tasks concurrently and returns sorted delays |

## Requirements

- Python 3.9 on Ubuntu 20.04 LTS
- pycodestyle 2.5
