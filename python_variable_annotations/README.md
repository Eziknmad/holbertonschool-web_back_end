# Python - Variable Annotations

This project covers type annotations in Python 3, using the `typing` module to annotate function parameters, return values, and module-level variables.

## Tasks

| File | Description |
|------|-------------|
| `0-add.py` | Annotated function that adds two floats |
| `1-concat.py` | Annotated function that concatenates two strings |
| `2-floor.py` | Annotated function that returns the floor of a float |
| `3-to_str.py` | Annotated function that converts a float to a string |
| `4-define_variables.py` | Annotated module-level variables |
| `5-sum_list.py` | Annotated function that sums a list of floats |
| `6-sum_mixed_list.py` | Annotated function that sums a list of ints and floats |
| `7-to_kv.py` | Annotated function that returns a key-value tuple |
| `8-make_multiplier.py` | Annotated function that returns a multiplier function |
| `9-element_length.py` | Annotated function that returns elements with their lengths |

## Requirements

- Python 3.9 on Ubuntu 20.04 LTS
- pycodestyle 2.5
